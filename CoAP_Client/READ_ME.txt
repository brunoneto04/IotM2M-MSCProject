1.É necessário instalar a biblioteca aiocoap.
2.É necessário alterar o IP no ficheiro 'define_ip' para o IP do servidor coap
3.Para correr os ficheiros basta usar o seguinte comando 
    python '.\<nome_do_ficheiro>.py'

Nota: Os pedidos estão criados para o proposito desejado. 
Os pedidos estão automatizados de uma forma e ordem que pode ser alterada.
Ajusta os pedidos como fizer mai sentido para o teu caso de uso.
